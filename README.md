# UAS-Kelompok-Mikrokontroler
Deskripsi Sistem Proyek ini mengusulkan "Sistem Jemuran Pintar Otomatis" yang dikendalikan oleh mikrokontroler berbasis ESP32. Sistem ini terintegrasi dengan sensor hujan dan sensor suhu/cahaya untuk membaca kondisi lingkungan sekitar.
